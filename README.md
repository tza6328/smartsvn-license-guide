## 几个日常用起来真香的功能

**标签和分支管理：不用再背那一长串 URL 了**

老 SVN 用户都知道，命令行切分支要手动输路径，搞错一个字符就重头来。SmartSVN 的标签浏览器把所有分支和标签按层级列出来，点选切换就行，干净利落。

**颜色标识一眼看状态**

项目目录里，红色箭头代表本地有修改，绿色箭头代表仓库有更新。一打开就知道哪些文件待处理，不用挨个点开检查。

**内置文件比较 + 冲突解决**

不用再额外装 WinMerge 或者 FileMerge 了。SmartSVN 内置的 Diff 工具支持语法高亮、行内差异对比，冲突解决也是三路合并，在提交窗口里就能直接看差异、写提交信息。

**Windows 资源管理器 / macOS Finder 集成**

文件右键菜单里直接就有 SVN 操作，图标会覆盖显示文件状态，不必每次都把主程序打开。
